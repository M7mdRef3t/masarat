المُوجَّهة

ابنِ مسارات تعافي ونمو تتكيّف مع المستخدم — خطوة بخطوة. جاهز للذكاء الاصطناعي. صفر معتمدات.

ابدأ الآن اكتشف المميزات
// 1. Determine the right path from diagnosis import { resolvePathId, buildFallbackRecoveryPath } from "@alrehla/masarat"; const pathId = resolvePathId({ zone: "red", contact: "high" }); // → "path_protection" (درع الحماية) // 2. Build a 3-week recovery path const path = buildFallbackRecoveryPath({ personLabel: "ماما", pathId }); // → { phases: { week1, week2, week3 }, tasks: [...] } // 3. Or generate a dynamic plan from patterns import { generateDynamicPlan } from "@alrehla/masarat"; const plan = generateDynamicPlan("ماما", "red", patterns, insights); // → Personalized 4-week plan with actions per pattern

ما الذي يقدّمه Masarat؟

🗺️

محلّل المسار الذكي

يحدد المسار المناسب تلقائياً من المنطقة (أحمر/أصفر/أخضر) + مستوى الاحتكاك + نوع الألم السائد.

📋

خطط 3 أسابيع جاهزة

5 مسارات مُنسّقة بعناية — كل مسار = 3 أسابيع × مهام يومية مخصصة مع معايير نجاح واضحة.

🧬

توليد خطط ديناميكية

يبني خطط مخصصة من أنماط مكتشفة — ذنب مفتعل، ضغط مالي، انتهاك حدود — كل نمط = استراتيجية دفاع.

⚡

مسار الأزمة السريع

جملة خروج فورية + تمرين تنفس + خطوة تالية — كلها في ثانية واحدة. للأزمة ومحتاج يتحرك دلوقتي.

🤖

جاهز للذكاء الاصطناعي

prompt templates مبنية مسبقاً — أدخل أي AI provider وابدأ توليد مسارات مخصصة فوراً.

🔌

صفر معتمدات

Pure TypeScript — لا React، لا Node، لا أي framework. يعمل في أي بيئة JavaScript.
